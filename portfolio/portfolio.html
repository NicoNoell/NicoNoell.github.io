My Programming-Languages

Here are the programming-languages I've learned over time together with my level of experience with them:

Python (experienced)

Python was my first real programming language (after Scratch). It was the basis for my understanding of object-oriented programming and has remained my favorite programming language since I first got into touch with it. Below or under Projects you will find many little programms and games made with Python.
I have around years of experience with Python (though I don't think that's a reliable measurement for coding-skill).


C# (intermediate)

Like almost every young developer I was mostly interested in making games and thus got into touch with C# and Unity. It was a considerable step up from the simple (more or less) typeless python I was used to and it took some time for me to be able to produce something without a tutorial. After a year or so I, for the time, quit Unity and worked more closely with Windows Forms in .NET (dot-Net).
Then, around 2 years later I again came into contact with C# at an internship. I learned concepts like Unit-testing (which I still often find superfluous) and navigating large code bases.
Overall, I have around years of experience with C#.


C++ (learning)

I have always been a bit scared of C++. It's a very powerful language that has evolved so much over the years, that at least I think, it cannot be fully understood by just one person. Currently I'm trying to learn the basics of it anyways, while building my own Ray-Tracer following this nice Article that offers a great introduction to newcommers like me.


Html, Css and Javascript (I did those to myself to build this Website)

As the summary-heading might give away, I'm not really a big fan of any of the three. I don't hate them, but they are often just not nice to work with (you dont want to know how often i needed to look up how to correctly center elements or text while building this website). If necessary however, I am able to create whatever is needed with them, but I'm already sure that I don't want to be a Web-Developer.


Java (just basic stuff)

The only time I've used Java so far was in High-School, where we made different data-structures and talked about concepts like recursion and inheritance. It was all very boring, expecially if you already know everything. Anyways, I don't really have an opinion on Java. All I know is its slogan: "Write once, debug run everywhere".
